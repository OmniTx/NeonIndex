# Welcome to NeonIndex! 👋

This is a demo README file showing what your file browser can display.

## Getting Started

Your files will appear in the table above. This README is rendered automatically!

## Supported Markdown

NeonIndex supports basic markdown formatting:

- **Bold text** and *italic text*
- `inline code` formatting
- [Links to websites](https://github.com)
- Bullet point lists
- Headers (H1 through H6)

## Quick Tips

- Upload files using the upload button (admin only)
- Click folder names to navigate inside
- Toggle between dark and light themes
- Access admin panel at `/admin.php`
